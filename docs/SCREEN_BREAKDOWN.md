# Suggestions Module - Screen-by-Screen Breakdown

## Design System Reference (from Figma)

### Colors
```css
--color-tint-primary: #ff5b24;        /* Vipps orange - primary actions */
--color-tint-secondary: #5e3dc2;      /* Purple - secondary links */
--color-avatar-surface: #ffe5db;      /* Peach - avatar background */
--color-background-primary: #ffffff;  /* White */
--color-surface-primary: #f7f7f7;     /* Light gray cards */
--color-label-primary: #000000;       /* Primary text */
--color-label-secondary: #8e8e93;     /* Secondary text */
--color-fill-primary: rgba(118, 118, 128, 0.12); /* Borders */
```

### Typography
```css
--font-family: 'SF Pro', -apple-system, BlinkMacSystemFont, sans-serif;
--font-size-title: 18px;
--font-size-body: 14px;
--font-size-caption: 12px;
--font-weight-medium: 500;
--font-weight-semibold: 600;
```

### Spacing
```css
--spacing-single: 8px;
--spacing-double: 16px;
--avatar-size: 56px;
--card-radius: 10px;
--avatar-radius: 999px; /* Fully round */
```
