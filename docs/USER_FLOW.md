# Suggestions Module - User Flow Documentation

## Overview

The Suggestions module provides an algorithm-generated list of people and merchants that users might want to send money to or request money from. Based on user research, the module prioritizes:
- **User control** - Easy pinning, hiding, and customization
- **Transparency** - Clear explanations for why someone is suggested
- **Trust** - Visible controls to manage the algorithm's output

1. **Screen Reader Support**
   - All avatars have alt text with person/merchant name
   - Pin status announced ("Anna Solvang, pinned favorite")
   - Why-suggested text read aloud

2. **Motor Accessibility**
   - Long-press has alternative tap + overflow menu
   - Edit mode provides larger touch targets
   - Reorder available via buttons, not just drag

3. **Visual Accessibility**
   - Pin badge has sufficient contrast
   - Why-suggested text meets WCAG 2.1 AA
   - Focus indicators visible in edit mode
